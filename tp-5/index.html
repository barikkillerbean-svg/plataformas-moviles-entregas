Control de Versiones con Git

Git es un sistema de control de versiones utilizado para manejar cambios en proyectos.

Comandos más usados

  • git init - Crear repositorio
  • git clone - Copiar repositorio
  • git status - Ver estado
Comando Función
git add Agregar archivos
git commit Guardar cambios

Git sirve para controlar versiones del código.
